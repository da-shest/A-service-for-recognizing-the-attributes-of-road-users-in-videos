# A-service-for-recognizing-the-attributes-of-road-users-in-videos
Сервис разрабатывается по заказу внешней организации на кафедре ИУ6 "Компьютерные системы и сети" МГТУ им. Н.Э. Баумана. Данный сервис является частью системы по видеоаналитеке дорожного движения. На данном этапе система извлекает атрибуты пешеходов и транпортных средств. В дайльнейшем будет разрабатываться анализ, поиск аномалий на основе извлеченных данных. Подробнее про сервис раписано в [Расчетно-пояснительной записке](РПЗ_ШестаковДА_ИУ6-33М.pdf).
